# Project Instructions

## Environment
- **Python Virtual Environment:** Always use the virtual environment located at `./venv_3.9/`.
- **Python Executable:** Use `./venv_3.9/bin/python3` for all Python-related commands.
- **Django Management:** Use `./venv_3.9/bin/python3 manage.py` for all management tasks.

## Settings
- **Default Settings Module:** Use `litloop_project.settings.dev` for local development and testing unless otherwise specified.
